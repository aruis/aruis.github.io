某种情况下，其实就想用Spring Boot提供的那种即开即用的开发体验，但是我真的对Spring保姆式的一揽子工程不怎么感冒。尤其是数据库JDBC这块。常见的Java系里提到的技术，我真的一个都不想用，我最钟爱的数据库类库其实就是groovy.sql.SQL，简单而强大，配合Groovy之后，再也没有繁琐的Java Bean和无边无际的get、set。
想了解更多groovy.sql.SQL欢迎查看官方文档：http://groovy-lang.org/databases.html
